# NLW#3 - Happy Challenge
![Badge](https://img.shields.io/badge/JavaScript-EcmaScript(5.1)-%9fb945) [ ![Badge](https://img.shields.io/badge/Handlebars.js-v4.7.6-%9fb945)](https://www.npmjs.com/package/handlebars) ![Badge](https://img.shields.io/badge/CSS-%9fb945)
Desafio da edição Next Week Level 3ª edição, realizado pelo site https://nextlevelweek.com, a proposta foi desenvolver um aplicativo de cadastro de instituições para abrigo de crianças em situação de vulnerabilidade social. Este aplicativo permite, cadastrar as instituições, visualizar as instituições cadastradas no mapa, visualizar as páginas de cada instituição e buscar no google maps a sua localização. Para utilizar esse POC, em ambiente de produção é necessário adaptá-lo para esse fim.

## O que fazer após clonar <a name="o-que-fazer"></a>

### Instalar o sqlite-async <a name="instalar-o-sqlite-async"></a>
```
 npm i sqlite-async
```  
### Abrir o terminal dentro do diretório do projeto <a name="abrir-terminal-dentro-diretorio"></a>
Rodar o seguinte comando no terminal.
```
npm start
```
Esse comando irá rodar o banco de dados na porta 5500, para visualizar o frontend, no navegador digitar:
``` localhost:5500      ``` ou  ```    127.0.0.1:5500 ```
